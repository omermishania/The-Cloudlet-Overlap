
### Goals
- The trainee will understand what kickstart is
- The trainee will know how to create a kickstart file

### Tasks
- Create a kickstart file, in this file define the following:
  - Root password
  - Network settings
  - Filesystem settings
  - Timezone
  - Enable the httpd service
